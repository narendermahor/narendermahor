# Narender Mahor Academic Website

This repository contains a static academic website for Narender Mahor. It is built with plain HTML and CSS so it can be hosted directly with GitHub Pages.

## Files

- `index.html` - main website content
- `styles.css` - responsive styling
- `assets/Narender_Mahor_CV.pdf` - downloadable CV
- `assets/research-banner.png` - homepage visual asset

## Preview Locally

Open `index.html` in a browser, or run a small local server from this folder:

```powershell
py -m http.server 8000
```

Then visit `http://localhost:8000`.

## Host on GitHub Pages

1. Create a GitHub repository named `narender-mahor.github.io` for a personal GitHub Pages site, or use any repository name for a project site.
2. Push these files to the repository's default branch.
3. In GitHub, go to `Settings` > `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select the default branch and the root folder.
6. Save. GitHub will publish the site after the Pages build finishes.

If the repository is named `narender-mahor.github.io`, the site will be available at `https://narender-mahor.github.io/`.
